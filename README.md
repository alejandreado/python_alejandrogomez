# Proyecto: Comprobador de Palíndromos + Tests Unitarios

Este proyecto implementa una función llamada `esPalindromo` que determina si
una frase es palíndroma ignorando tildes, mayúsculas, espacios y puntuación.

Incluye además una batería completa de tests automáticos usando `unittest`.

## Cómo ejecutar el proyecto

Crear el entorno virtual:

python -m venv .venv


Activarlo:

..venv\Scripts\activate


Ejecutar el programa:

python -m app.main


Ejecutar los tests:


python -m unittest discover -v
